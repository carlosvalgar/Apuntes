Primer párrafo

Este es el primer párrafo. El titulo de arriba esta creado con una etiqueta <h2> y este texto con una etiquetaueta <p> de párrafo. Ambos están dentro de un elemento <div> , al cual cambiaremos el color de fondo (background-color), borde (border) y relleno (padding). El texto del párrafo está justificado.




Segundo párrafo

Este es el segundo párrafo. El título de arriba está creado con una etiqueta <h2> y este texto con una etiqueta <p> de párrafo. En este caso no están agrupados con un elemento <div>. El título está separado 3 espacios con respecto al párrafo anterior y le hemos puesto un borde sólido, y cambiado el color de fuente y de fondo. Al párrafo le hemos aplicado la profiedad margin por la izquierda y tambien está justificado. Para resaltar esta frase, hemos utilizado una etiqueta <span> (fuente arial).
